# Java-BMI-Calculator

A simple Java console application that calculates the Body Mass Index (BMI) based on the user's weight and height.

## Features

* Calculate Body Mass Index (BMI).
* Classify BMI into:

  * Underweight
  * Normal Weight
  * Overweight
  * Obese
* Calculate the ideal weight based on the user's height.

## Technologies Used

* Java
* NetBeans IDE

## How to Run

1. Open the project in NetBeans or any Java IDE.
2. Run the program.
3. Enter your weight in kilograms (kg).
4. Enter your height in meters (m).
5. The program will display your BMI, BMI category, and ideal weight.
